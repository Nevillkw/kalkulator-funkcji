# Zgłaszanie podatności (Security)

Jeśli znalazłeś podatność:

- Nie publikuj jej publicznie w Issues
- Użyj Security Advisories (jeśli dostępne) lub skontaktuj się prywatnie: https://arturmrowicki.pl
- Dołącz minimalny opis, kroki reprodukcji i zakres wpływu

Uwaga: nie prowadzę formalnego programu bug bounty, a czasy reakcji nie są gwarantowane. Dziękuję za odpowiedzialne zgłoszenie.
